# Attribution and provenance

This project is an independent, focused implementation based on the feature and
license baseline of [xh321/LiteLoaderQQNT-Anti-Recall 0.3.0](https://github.com/xh321/LiteLoaderQQNT-Anti-Recall/tree/0.3.0), licensed under MIT.

Compatibility behavior was compared with the public documentation of
[MeiYongAI/QQNT-Toolbox 0.6.3](https://github.com/MeiYongAI/QQNT-Toolbox/releases/tag/v0.6.3).
No AGPL source code from that project is included in this distribution.

The optional loader runtime is distributed separately in `vendor/`:

- LiteLoaderQQNT 1.4.1, MIT, from the official GitHub release.
- `dbghelp_x64.dll` 1.1.2 from the official QQNTFileVerifyPatch release. It is an unsigned compiled loader bridge and is not part of the plugin source.
